Introduction

This is tool designed to store and share trade data from the Elite: Dangerous game.

My intention is to produce an application that's the middle ground between offline note-taking tools and the crowd source Elite Dangerous Data Network apps that are available.

The idea is that you and a few friends will each use this tool and have each copy pointed towards a common online source. This will allow you to individually record market supply and demand data and automatically make that information available amongst yourselves without making your valuable trade routes public knowledge.

Installation

The python script should work out of the box (although I've only tested it on Windows), provided you have the python path etc. all set up. Alternatively, you can convert the tool into a portable windows executable by following this useful tutorial.

The latter option is how I intended the tool to be used and the finished article will be distributed as such an executable.

Current State

If you've had a look at the contents of the repository at all, you'll see that this thing is definitely still a work-in-progress. So here's the plan:

  • Swap the commodity text box with a quicker way to enter supply/demand data (possibly radio buttons).
  • Suggest trade routes based on available data.
  • Read from/Save to an online source (probably a text file on Google Drive) on start/exit.
